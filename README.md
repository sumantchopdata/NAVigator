# NAVigator
A Telegram Bot that gives you insights about the mutual fund of your interest. Just type in the name of the fund and this bot make an API call to mfapi.in to give you the latest statistics.

Steps:
 1. Create a new virtual environment and install the packages mentioned in the requirements.txt file.
 1. Create a telegram bot using the instructions in the file telegram_bot_instructions.txt.
 1. Activate your virtual environment in the terminal and run the following command: python bot_runner.py
 1. Now write the full name of the mutual fund you're interested in.
 1. The bot does its magic and fetches for you the latest insights about the fund!
